# Class-order

## Install
* install [zeromq](https://github.com/zeromq/zeromq.js/)
* install [opencv4nodejs](https://github.com/justadudewhohacks/opencv4nodejs)


```
$ cd webserver
$ npm install
```

## Before open server,we need to get class.

```
$ cd webserver
$ node get_class
```

## Open web server

```
$ cd webserver
$ npm install 
$ node app.js
```

## Open zeromq server

```
$ cd ./webserver/model
$ node respond
```
